**SLCampusUltra** is a web application built with ASP.NET WebForms and C#, designed to serve as a multi-purpose platform for campus life. Currently, it features a fully functional **Online Grocery Shop** module, with plans to expand into other campus-related services.

## 📚 About The Project

SLCampusUltra aims to simplify daily life for students and campus communities by providing essential online services. The initial focus is on a robust grocery ordering system, allowing users to browse products, manage their shopping, and handle orders seamlessly.

### ✨ Current Features

*   **🛒 Online Grocery Shop**
    *   Browse and search for grocery items.
    *   Product management interface (`ManageProducts.aspx`).
    *   Dedicated search functionality (`Search1.aspx`).
    *   Shopping cart and order placement logic (implemented in code-behind files).
*   **🏫 Campus Integration Foundation**
    *   Built with future expansion in mind to include other campus-specific services.
    *   Modular ASP.NET WebForms architecture.

### 🚧 Planned Features

*   Campus event listings and management.
*   Student marketplace for books and supplies.
*   Food delivery or campus dining integration.
*   User authentication and role-based access (Students, Admins, Vendors).
